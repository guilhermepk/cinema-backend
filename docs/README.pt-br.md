# Cinema Backend

> [English version](../README.md)

> PROJETO DE ESTUDO

Este projeto é o backend de um sistema de `vendas de ingressos de uma rede de cinemas`. O desafio principal do projeto é lidar com `concorrência`: múltiplos usuários tentando comprar os mesmos assentos simultaneamente.

A ideia é estudar o desenvolvimento de `sistemas distribuídos` que utilizam `comunicação assíncrona` por `mensageria` e `orientação a eventos` com `NestJS` e `Kafka`.

> Para saber mais, leia sobre [o desafio que este projeto visa cumprir](challenge.pt-br.md)
