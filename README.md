# Cinema Backend

> [Leia aqui a versão em Portugês do Brasil](./docs/README.pt-br.md)

> STUDY PROJECT

This project is the backend of a `ticket sales system for a cinema chain`. The project's main challenge is dealing with `concurrency`: multiple users trying to buy the same seats simultaneously.

The idea is to study the development of `distributed systems` that use `asynchronous communication` through `messaging` and `event-driven` with `NestJS` and `Kafka`.
